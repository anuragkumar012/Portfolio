## 📖 Overview

This repository contains the source code for the personal portfolio website of **Anurag Kumar**, a Data Scientist & AI Engineer. The website is designed with a premium, sleek dark mode theme featuring fluid grid layouts, interactive elements, custom ambient glow effects, and modern editorial typography. 

It serves as a professional showcase of Anurag's capabilities, projects, blog posts, and core technical skills.

---

## 🎨 Design & Aesthetic Highlights

- **Ambient Glow Effects:** Sleek, dynamic radial gradient highlights in the background that create depth and a high-end feel.
- **Editorial Typography:** A curated pairing of the classic, high-contrast serif font [Playfair Display](https://fonts.google.com/specimen/Playfair+Display) for titles/headers and the clean, geometric sans-serif font [Outfit](https://fonts.google.com/specimen/Outfit) for copy and badges.
- **Premium Dark Mode:** Custom deep backgrounds with harmonized borders, interactive service badges, and micro-interactions.
- **Fully Responsive:** Perfectly optimized for mobile, tablet, and desktop viewing.

---

## 🛠️ Technology Stack

- **Framework:** [Next.js 16.2.7 (App Router)](https://nextjs.org/) — Utilizing server-side rendering, optimized image handling (`next/image`), and layout structures.
- **Frontend Library:** [React 19.2.4](https://react.dev/) — Exploiting the latest React features for rendering speed and state management.
- **Styling:** [Tailwind CSS v4.0](https://tailwindcss.com/) + `@tailwindcss/postcss` — Leveraging the speed and design token systems of Tailwind's newest generation.
- **Icons:** [Lucide React](https://lucide.dev/) — High-quality, customizable svg icons.
- **Programming Language:** [TypeScript](https://www.typescriptlang.org/) — Fully type-safe development environment.

## 🚀 Getting Started

To run a copy of this project locally, follow these simple setup steps:

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed (recommended version `v18` or higher).

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/Protiflio.git
   cd Protiflio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

### Running the Application

1. **Start the development server:**
   ```bash
   npm run dev
   ```
   
2. **Open the site:**
   Open your browser and navigate to [http://localhost:3000](http://localhost:3000) to view the application.

### Production Build

To build the application for production deployment:

```bash
npm run build
```

To run the production build locally:

```bash
npm run start
```

## ☁️ Deployment

The easiest way to deploy this Next.js project is using the Vercel platform:

1. Push your code to a GitHub, GitLab, or Bitbucket repository.
2. Go to [Vercel](https://vercel.com/) and create a new project.
3. Import your repository and click **Deploy**. Vercel will automatically detect Next.js and build/host the project.
